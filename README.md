## PYCRAFTCO

**pycraftco** is a Python library that serves as a convenient wrapper for [Craft API](https://api.craft.co/docs/v1)

### Installation

You can install package via `pip`:

    $ pip install git+https://github.com/craft-machine/pycraftco.git


### Usage

```
MINI TUTORIAL
```
